
### Temperature Settings
| Asset type | Temperature | Why |
|-----------|-------------|-----|
| App/Logo icon | 0.6–0.8 | Consistent, controlled |
| Hero/Splash | 1.0–1.2 | Creative, dramatic |
| UI Background | 0.4–0.6 | Subtle, predictable |
| Social/Promo | 1.0–1.4 | Most creative freedom |
| Favicon | 0.4–0.6 | Ultra simple |

### Aspect Ratio Settings
| Asset type | Aspect ratio |
|-----------|-------------|
| Logo icon | 1:1 |
| Website hero | 16:9 |
| UI Background | 16:9 |
| IG post | 1:1 |
| IG story / WA status | 9:16 |
| Favicon | 1:1 |
| Proposal header | 16:9 |

### Workflow per Asset
1. Set **System Instructions** (copy from above — do this ONCE)
2. Set **Temperature** per asset type (see table)
3. Set **Aspect Ratio** per asset type (see table)
4. Paste the prompt
5. Generate — review — if 80% good, refine with follow-up: "same but [adjustment]"
6. Save winners with prompt number (e.g., `A4_winner.png`)
